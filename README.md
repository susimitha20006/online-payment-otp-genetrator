# Secure OTP-Based ATM Authentication System

## Overview
This project enhances ATM security by adding OTP verification during cash withdrawal.

## Features
- PIN + OTP authentication
- Prevents unauthorized withdrawals
- Secure transaction validation

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## How It Works
1. User enters ATM PIN.
2. System generates OTP.
3. OTP is sent to registered mobile.
4. Transaction completes only after OTP verification.
